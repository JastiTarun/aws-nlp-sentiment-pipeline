# AWS NLP Sentiment Pipeline

End-to-end AWS data pipeline for sentiment analysis of Amazon reviews.

## Architecture
S3 → Glue → SageMaker → Athena → QuickSight

## Services Used
- Amazon S3
- AWS Glue (Crawler + ETL)
- Amazon SageMaker
- AWS Athena
- Amazon QuickSight

## Outcome
Generated sentiment predictions and built analytics dashboard.
